# Company Explorer

A responsive React application built as part of a frontend assignment to explore and filter companies using static JSON data.

## 🚀 Features

- Card-based company listing
- Search by company name
- Filter by location and industry
- Responsive layout
- Loading and empty states

## 🛠 Tech Stack

- React (Vite - latest)
- Bootstrap
- Vanilla CSS
- Static JSON data
